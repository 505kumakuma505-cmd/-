# PixL & VoiD FX 代用アプリ プロジェクト

## 概要
FortniteモバイルのFPS設定を変更するためのPixLとVoiD FXの代用アプリケーション

## 構成
1. **PixL Store代用** - アプリ管理・APK配布機能
2. **VoiD FX代用** - ゲーム設定変更機能

## 対応FPS
30, 60, 120, 144, 165, 180, 200, 240, 360 FPS
